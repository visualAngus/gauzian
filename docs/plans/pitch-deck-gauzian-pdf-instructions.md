# Instructions — Export PDF du Pitch Deck

## Contexte

Pandoc n'est pas disponible dans l'environnement actuel. Voici les instructions pour générer le PDF localement.

---

## Option 1 — Pandoc (recommandée)

### Installation

```bash
# macOS
brew install pandoc

# Ubuntu/Debian
sudo apt-get install pandoc

# Fedora
sudo dnf install pandoc
```

### Commande pour générer le PDF

```bash
# Avecxelatextemplate
pandoc /home/gael/Bureau/gauzian/pitch-deck-gauzian.md \
  -o /home/gael/Bureau/gauzian/gauzianMain/docs/plans/pitch-deck-gauzian.pdf \
  --from markdown \
  --to pdf \
  --pdf-engine=xelatex \
  -V mainfont="Helvetica Neue" \
  -V sansfont="Arial" \
  -V geometry:margin=1in
```

### Alternative avec pdflatex

```bash
pandoc /home/gael/Bureau/gauzian/pitch-deck-gauzian.md \
  -o /home/gael/Bureau/gauzian/gauzianMain/docs/plans/pitch-deck-gauzian.pdf \
  --pdf-engine=pdflatex \
  -V mainfont="DejaVuSans"
```

## Option 2 — Via VS Code

1. Installer l'extension **Markdown PDF**
2. Ouvrir `pitch-deck-gauzian.md`
3. Clic droit → **Markdown PDF: Export (pdf)**

---

## Option 3 — Via Docker

```bash
docker run --rm \
  -v /home/gael/Bureau/gauzian:/data \
  pandoc/latex \
  /data/pitch-deck-gauzian.md \
  -o /data/gauzianMain/docs/plans/pitch-deck-gauzian.pdf \
  --pdf-engine=xelatex
```

---

## Recommandations

- **Police** : Helvetica Neue, Arial, ou DejaVuSans (fallback)
- **Marges** : 1 inch (2.54 cm) pour une meilleure lisibilité
- **Option slide** : Pour un format présentation, utiliser `beamer` avec `--to beamer`
